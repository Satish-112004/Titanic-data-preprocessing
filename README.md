# Titanic Data Preprocessing and Analysis Project

## 📌 Objective
The goal of this project is to clean and preprocess the Titanic dataset and analyze survival patterns.

## 🧹 Data Preprocessing
- Handled missing values:
  - Age filled with mean
  - Embarked filled with mode
- Dropped unnecessary columns:
  - Cabin, Ticket, PassengerId, Name
- Converted categorical data:
  - Sex → 0,1
  - Embarked → 0,1,2

## 📊 Data Analysis
- Females had higher survival rate
- Higher class passengers survived more

## 🤖 Model
- Used Logistic Regression
- Achieved good accuracy

## 🛠 Tools Used
- Python
- Pandas
- Seaborn
- Scikit-learn

## 📁 Files
- Titanic notebook (.ipynb)

## 🚀 Conclusion
The dataset was cleaned and used to build a basic predictive model.
